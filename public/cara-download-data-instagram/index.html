Cara Download Data Instagram JSON untuk Cek Unfollowers

Untuk cek siapa yang tidak follow back Instagram, Anda perlu mengambil data resmi dari Instagram dalam format JSON. File yang dibutuhkan adalah followers_1.json dan following.json.

Langkah download data Instagram

  1. Buka Instagram, lalu masuk ke Settings and privacy.
  2. Pilih Accounts Center.
  3. Buka Your information and permissions.
  4. Pilih Download your information.
  5. Buat request download untuk akun Instagram yang ingin dicek.
  6. Pilih Select types of information.
  7. Centang hanya Followers and following.
  8. Pilih format JSON, bukan HTML.
  9. Submit request dan tunggu sampai file download tersedia.
  10. Download file ZIP, ekstrak, lalu cari folder followers_and_following.
  11. Gunakan file followers_1.json dan following.json di instalyze.

Kenapa harus format JSON?

Format JSON lebih mudah dibaca aplikasi analyzer. Dengan file JSON, instalyze bisa membandingkan followers dan following untuk menampilkan akun yang tidak follow back, akun yang belum Anda follow balik, dan mutual followers.

Apakah data aman?

instalyze memproses file di browser. File JSON yang Anda pilih tidak diupload ke server, sehingga data followers dan following tetap berada di perangkat Anda.
